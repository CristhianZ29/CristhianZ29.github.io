Gestión de Memoria

Conceptos clave para evaluaciones.

Memoria Virtual

  • Extiende la RAM usando disco
  • Permite multitarea
  • Puede causar lentitud (swap)
  • Simula más memoria

Paginación

  • Bloques fijos (páginas)
  • Evita fragmentación externa
  • Page Fault afecta rendimiento
  • Uso eficiente de memoria

Segmentación

  • Bloques lógicos
  • Tamaño variable
  • Mejor organización
  • Puede fragmentar memoria

Rendimiento

Estos mecanismos optimizan el uso de memoria y mejoran la ejecución de procesos. Herramientas como RAMMap muestran memoria en uso, espera y libre, evidenciando cómo el sistema mejora el rendimiento.
