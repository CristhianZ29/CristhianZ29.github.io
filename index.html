Historia de los Sistemas Operativos

Los sistemas operativos surgieron como una solución para facilitar la interacción entre el usuario y el hardware del computador. En las primeras generaciones de computadoras los programas se ejecutaban directamente en el hardware, lo que hacía que el proceso fuera complejo y poco eficiente.

Con el paso del tiempo aparecieron sistemas de procesamiento por lotes que permitían ejecutar varios programas de manera automática. Posteriormente surgieron sistemas multiprogramados que podían mantener varios procesos en memoria al mismo tiempo.

Actualmente los sistemas operativos gestionan recursos como memoria, CPU, almacenamiento y dispositivos de entrada y salida, permitiendo que múltiples aplicaciones funcionen simultáneamente de forma segura y eficiente.

Algoritmos de Planificación de Procesos

La planificación de procesos es el mecanismo que utiliza el sistema operativo para decidir qué proceso debe ejecutarse en el procesador y durante cuánto tiempo. Esto es fundamental para lograr eficiencia y buen rendimiento del sistema.

  • FCFS (First Come First Served): ejecuta procesos en el orden en que llegan.
  • SJF (Shortest Job First): ejecuta primero los procesos con menor tiempo de ejecución.
  • Round Robin: asigna un pequeño intervalo de tiempo a cada proceso para evitar monopolizar el CPU.
  • Prioridad: los procesos con mayor prioridad se ejecutan primero.

Cada algoritmo tiene ventajas y desventajas dependiendo del tipo de sistema y del objetivo que se quiera optimizar, como tiempo de respuesta o uso eficiente del procesador.

Instalación básica de Debian Linux

Debian es una distribución de Linux reconocida por su estabilidad y seguridad. Es ampliamente utilizada en servidores y entornos académicos.

Pasos básicos de instalación:

  1. Descargar la imagen ISO desde la página oficial.
  2. Crear un USB booteable.
  3. Iniciar el computador desde el USB.
  4. Seleccionar el tipo de instalación.
  5. Configurar idioma, zona horaria y teclado.
  6. Crear usuario y contraseña.
  7. Configurar particiones del disco.
  8. Instalar el sistema base.

Comandos básicos después de instalar Debian:

sudo apt update
sudo apt upgrade
sudo apt install sudo

Ciclo de vida de los procesos

Un proceso es un programa en ejecución. Durante su funcionamiento pasa por diferentes estados que determinan su relación con el procesador y los recursos del sistema.

  • Nuevo: el proceso acaba de crearse.
  • Listo: está preparado para ejecutarse.
  • En ejecución: está utilizando el CPU.
  • En espera: espera algún recurso o evento.
  • Finalizado: el proceso termina su ejecución.

El sistema operativo toma decisiones constantemente para mover procesos entre estos estados, buscando mejorar la eficiencia y mantener el sistema estable.
